## 📖 Project Overview

Air-Aware is a modern React-based web application designed to educate users about air pollution, its causes, health impacts, and preventive measures.

The platform combines an editorial-style user interface with an AI-powered chatbot that helps users understand environmental issues through interactive conversations.

The website delivers an engaging reading experience while promoting awareness about climate change and air quality.

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/B-Praveen18/Air-Aware.git
```

Move into the project

```bash
cd Air-Aware
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```
